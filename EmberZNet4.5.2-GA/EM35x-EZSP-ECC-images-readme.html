EM35x EZSP images containing the ECC library

Overview

This release is meant to be used with EmberZNet 4.5.x release and contains the EZSP images for EM35x that are built with the ECC Library from Certicom

Software Versions
  • stack software -- please refer to the (stack) release note for the build number
  • ECC Library - version 1.0.1.20

    EZSP images
    EM35x EZSP images provided support all optional libraries. See the table at the end of this document for a description of these optional libraries. EBL files can be loaded using the bootloader or em3xx_load - the tokens are not erased during the load.

  • em35x-ncp-spi-use-with-ezsp-spi-bl-ecc-45xx.ebl -- This image is for use with a SPI connected host. Can run as any type of ZigBee device. This image is intended to be run with the ezsp-spi bootloader (found in tool/bootloader-em35x directory as part of stack installer). The image for em357, em357-ncp-spi-use-with-ezsp-spi-bl-ecc-45xx.ebl, uses a 32KB SimEEPROM - it supports 50 binding table and link key table entries each. The image for em351, em351-ncp-spi-use-with-ezsp-spi-bl-ecc-45xx.ebl, uses a 8KB SimEEPROM - it supports 32 binding table and 12 link key table entries.
  • em35x-ncp-spi-with-ezsp-spi-bl-ecc-45xx.hex -- This image is for use with a SPI connected host. Can run as any type of ZigBee device. This image includes ezsp-spi bootloader. The image for em357, em357-ncp-spi-with-ezsp-spi-bl-ecc-45xx.hex, uses a 32KB SimEEPROM - it supports 50 binding table and link key table entries each. The image for em351, em351-ncp-spi-with-ezsp-spi-bl-ecc-45xx.hex, uses a 8KB SimEEPROM - it supports 32 binding table and 12 link key table entries.
  • em35x-ncp-uart-rts-cts-use-with-serial-uart-bl-ecc-45xx.ebl -- This image is for use with a UART connected host using hardware flow control (RTS/CTS). Can run as any type of ZigBee device. This image is intended to be run with the serial-uart bootloader (found in tool/bootloader-em35x directory as part of stack installer). The image for em357, em357-ncp-uart-rts-cts-use-with-serial-uart-bl-ecc-45xx.ebl, uses a 32KB SimEEPROM - it supports 50 binding table and link key table entries each. The image for em351, em351-ncp-uart-rts-cts-use-with-serial-uart-bl-ecc-45xx.ebl, uses a 8KB SimEEPROM - it supports 32 binding table and 12 link key table entries.
  • em35x-ncp-uart-rts-cts-with-serial-uart-bl-ecc-45xx.hex -- This image is for use with a UART connected host using hardware flow control (RTS/CTS). Can run as any type of ZigBee device. This image includes serial-uart bootloader. The image for em357, em357-ncp-uart-rts-cts-with-serial-uart-bl-ecc-45xx.hex, uses a 32KB SimEEPROM - it supports 50 binding table and link key table entries each. The image for em351, em351-ncp-uart-rts-cts-with-serial-uart-bl-ecc-45xx.hex, uses a 8KB SimEEPROM - it supports 32 binding table and 12 link key table entries.
  • em35x-ncp-uart-xon-xoff-use-with-serial-uart-bl-ecc-45xx.ebl -- This image is for use with a UART connected host using hardware flow control (XON/XOFF). Can run as any type of ZigBee device. This image is intended to be run with the serial-uart bootloader (found in tool/bootloader-em35x directory as part of stack installer). The image for em357, em357-ncp-uart-xon-xoff-use-with-serial-uart-bl-ecc-45xx.ebl, uses a 32KB SimEEPROM - it supports 50 binding table and link key table entries each. The image for em351, em351-ncp-uart-xon-xoff-use-with-serial-uart-bl-ecc-45xx.ebl, uses a 8KB SimEEPROM - it supports 32 binding table and 12 link key table entries.
  • em35x-ncp-uart-xon-xoff-with-serial-uart-bl-ecc-45xx.hex -- This image is for use with a UART connected host using hardware flow control (XON/XOFF). Can run as any type of ZigBee device. This image includes serial-uart bootloader. The image for em357, em357-ncp-uart-xon-xoff-with-serial-uart-bl-ecc-45xx.hex, uses a 32KB SimEEPROM - it supports 50 binding table and link key table entries each. The image for em351, em351-ncp-uart-xon-xoff-with-serial-uart-bl-ecc-45xx.hex, uses a 8KB SimEEPROM - it supports 32 binding table and 12 link key table entries.
